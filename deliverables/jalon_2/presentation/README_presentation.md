# Présentation finale - Jalon 2

La présentation PDF finale du jalon 2 doit être placée dans ce dossier sous le nom :

```text
JALON_2_Presentation_Aftermath.pdf
```

Ce nom court permet d'identifier clairement le rendu oral dans le paquet `deliverables/jalon_2/`.

Une exportation PDF locale peut avoir un nom plus détaillé avec les noms des membres de l'équipe. Pour le rendu final du dépôt, utiliser le nom attendu ci-dessus.
